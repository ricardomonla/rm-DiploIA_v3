3. **Sincronización automática:**
   - Las funciones de `app-data.js` pueden sincronizar automáticamente la versión global si se configura así.
   - La versión global en `app-version.js` y las versiones individuales en `app-data.json` están vinculadas pero independientes.

4. **Ejecución de ejemplo:**
   ```bash

   node -e "import('./app-data/app-data.js').then(m => m.updateFileVersion('app-jsx', '1.0.2').then(() => console.log('Versión actualizada')))"
   ```

[VERSIONING STRATEGY]
- **Versión Global (app-version.js):** Refleja el estado general de la aplicación.
- **Versiones Individuales (app-data.json):** Reflejan el estado de cada archivo.
- **Sincronización:** Opcional, controlada por las funciones de `app-data.js`.
